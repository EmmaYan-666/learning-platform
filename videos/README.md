# 教学视频目录

这个目录用于存放睿丁欣研英语课程的教学视频文件。

## 📁 文件命名规范

请按以下格式命名视频文件：

- `S1.mp4` - 第一课教学视频
- `S2.mp4` - 第二课教学视频
- `S3.mp4` - 第三课教学视频
- ...

## 📹 视频格式要求

- **格式**：MP4（推荐）
- **编码**：H.264
- **分辨率**：建议 720p 或 1080p
- **大小**：建议单个视频不超过 100MB

## 🔗 使用说明

1. 将视频文件命名为 `S1.mp4`、`S2.mp4` 等
2. 放入此 `videos/` 目录
3. 提交到 GitHub：
   ```bash
   git add videos/
   git commit -m "添加教学视频"
   git push
   ```
4. 视频会自动在学习平台中播放

## ⚠️ 注意事项

- GitHub 单文件限制 100MB，超过需要使用 Git LFS
- 建议压缩视频以加快加载速度
- 可以使用 HandBrake 等工具压缩视频
