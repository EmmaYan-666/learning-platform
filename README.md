# 🎓 儿童学习平台 (Kids Learning Platform)

一个为 5 岁儿童设计的互动学习平台，包含英语、汉字、字母等多个学习模块。

## 🌟 在线访问

**GitHub Pages**: https://emmayan-666.github.io/learning-platform/

## 📱 功能模块

### 1. 🔤 字母游戏
- 26 个英文字母学习
- 大小写显示
- 字母识别练习
- 语音朗读功能

### 2. 📚 单词游戏
- 动物、食物、家庭、颜色、身体等多个类别
- 50+ 常用英语单词
- 图片选择练习
- 错题本系统
- 进度追踪

### 3. 🈶 汉字游戏
- 《四五快读》第一册 50 个汉字
- 分 6 课，每课 8 字
- 拼音标注
- 高清图片配图
- 点击发音

### 4. 📖 句子跟读
- 50 个常用句子
- 拼音标注
- 语音朗读（0.8 倍速）
- 手动翻页
- 进度指示器

### 5. 📗 睿丁欣研英语课程（v2.3.1 新增）
- 课程管理（录入、编辑、删除）
- 单词和短语学习
- 视频播放功能
- 智能扩展周边词汇
- 学习进度追踪

## ✨ 核心特性

### 🎯 学习功能
- **语音朗读**：美音女声发音，支持调节速度
- **错题本**：自动记录错误，错题高频出现
- **进度追踪**：记录学习次数、正确率
- **激励系统**：星星奖励、连胜统计

### 🎨 个性化配置
- **Emoji 自定义**：点击"配置图片"按钮自定义单词图片
- **语音设置**：可调节语速、音调
- **进度保存**：所有进度自动保存到本地

### 📊 数据统计
- 正确次数、错误次数统计
- 总分和连胜记录
- 各模块进度条显示

## 🛠️ 技术栈

- **前端**：单 HTML 文件架构
- **存储**：localStorage 本地存储
- **语音**：Web Speech API
- **图片**：Emoji + Unsplash API
- **部署**：GitHub Pages

## 📁 项目结构

```
learning-platform/
├── index.html                 # 主学习平台（v2.3.1）
├── word_game_v3.html         # 单词游戏独立版
├── versions/                  # 历史版本
│   ├── learning_platform_v2.0.html
│   ├── learning_platform_v2.1_chinese_optimized.html
│   └── learning_platform_v2.2_optimized.html

│   └── *.docx                # 可打印单词卡片
└── deploy/                   # 部署文件
```

