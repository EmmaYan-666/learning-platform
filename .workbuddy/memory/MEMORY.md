# 工作记忆 - 英语单词学习系统

## 项目概述

为5岁儿童创建英语单词学习系统，包含：
- 单词卡片生成系统
- 短语和例句文档
- 趣味互动练习
- 持续更新的单词数据库

## 系统架构

### 核心脚本文件
- `word_database.json` - 单词数据库（53个单词）
- `word_list_manager.py` - 单词管理模块
- `generate_picture_cards.py` - 生成图片卡片（横向，每页8张）
- `generate_cards.py` - 生成标准卡片（竖向）
- `generate_phrases_optimized.py` - 生成短语例句（表格布局）
- `generate_interactive_quiz.py` - 生成互动练习

### 单词分类
- ANIMALS：9个单词
- FOOD：12个单词
- FAMILY：5个单词
- COLORS：8个单词
- ADJECTIVES：4个单词
- BODY：15个单词（新增）

### 文档特点
- 单词卡片：横向排版，每页8张，优化空间利用
- 短语例句：表格布局，页数少，字体大
- 练习题：互动游戏，勾选、连线、分类

## 在线互动学习平台

### 功能特性（word_game_v3.html - 最新版本）

**核心功能：**
- 语音朗读：美音女声发音，点击单词播放
- 四选一游戏：显示4张图片（emoji）供选择
- 即时反馈：正确/错误动画效果

**🎨 Emoji配置功能（V3新增）：**
- 点击"配置图片"按钮，打开配置面板
- 可以修改每个单词对应的emoji
- 配置自动保存，刷新页面不会丢失
- 可以恢复默认设置
- 方便家长/老师根据孩子喜好自定义图片

**错题本系统：**
- 自动记录错误单词
- 错误次数统计
- 错题出现频率提升（错得越多出现越频繁）
- 右侧面板查看错题本

**激励系统：**
- 正确答案：三颗星星逐个点亮 + 语音鼓励（Excellent!等）
- 错误答案：鼓励性提示 + 显示正确答案
- 统计数据：正确次数、错误次数、总分、连胜

**用户体验改进：**
- 错误后停留页面，需手动点击"下一题"
- 图片辨识度优化（Head: 🧠, Eye: 👀, Shoulder: 🏋️ 等）
- 音量可调节
- 进度自动保存
- emoji可自定义配置

**单词分类：**
- 🐾 动物（9个）
- 🍔 食物（12个）
- 👨‍👩‍👧‍👦 家庭（5个）
- 🎨 颜色（8个）
- 👂 身体（15个）
- 📏 形容词（4个）

## 最新进展 (2026-03-25)

### 学习平台三大功能修复与优化

**修复内容**：

1. **句子跟读扩容与随机化**：
   - 句子数据从27句扩充到50句
   - 每课从4-5句增加到8-9句
   - 实现随机显示句子(每次切换课程或完成一轮都会重新随机打乱)
   - 提升学习效果,避免机械记忆

2. **首页进度条功能实现**：
   - 添加localStorage进度持久化
   - 实时记录各游戏的正确/错误次数
   - 计算正确率并更新进度条显示
   - 返回首页时自动更新进度条
   - 页面刷新后进度不丢失

3. **图片配置功能**：
   - 为字母、单词、汉字三个游戏添加"配置图片"按钮
   - 实现emoji自定义配置面板
   - 配置自动保存到localStorage
   - 支持恢复默认设置
   - 方便家长/老师根据孩子喜好调整图片

**技术实现**：
- localStorage存储：`learningProgress`(进度数据)、`learningPlatformEmojis`(emoji配置)
- 随机算法：Fisher-Yates洗牌算法
- 数据结构优化：为每个游戏类型添加独立的配置管理
- UI组件：右侧滑出式配置面板,支持滚动浏览

**用户体验改进**：
- 句子学习不再枯燥,每次都有新鲜感
- 进度可视化,激励持续学习
- 图片可定制,适应不同孩子的认知偏好
