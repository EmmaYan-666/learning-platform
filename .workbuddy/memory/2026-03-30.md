## Bug修复（v2.3.1）

### 问题1：语音设置面板打不开

**原因**：
- 语音设置面板使用`style.display`控制显示
- 但遮罩层CSS使用`.show`类控制
- 导致遮罩层不显示，面板无法正常打开

**解决方案**：
1. 修改`openVoiceSettings()`函数，使用`classList.add('show')`替代`style.display`
2. 修改`closeVoiceSettings()`函数，使用`classList.remove('show')`
3. 移除语音设置面板的内联`style="display: none;"`属性

### 问题2：英语课程中单词+短语整合未生效

**原因**：
- 课程数据从localStorage加载
- localStorage中存储的是旧版本数据结构
- 导致新的渲染逻辑无法正确显示

**解决方案**：
- 移除localStorage优先级，直接使用代码中的课程数据
- 确保每次都使用最新版本的课程结构

**修改代码**：
```javascript
// 旧代码（问题）
let readingCourses = JSON.parse(localStorage.getItem('readingCourses')) || {...}

// 新代码（修复）
let readingCourses = {...}  // 直接使用最新数据
```
