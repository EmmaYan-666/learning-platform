### 技术细节

**新增localStorage键**：
- `learningProgress`: 存储各游戏进度
- `learningPlatformEmojis`: 存储自定义emoji配置

**核心函数**：
- `loadProgress()` / `saveProgress()`: 进度管理
- `updateHomeProgress()`: 更新首页进度条
- `toggleConfigPanel()`: 配置面板开关
- `getCustomEmoji()`: 获取自定义emoji

**数据结构**：
```javascript
learningProgress = {
  alphabet: { correct: 0, wrong: 0 },
  word: { correct: 0, wrong: 0 },
  chinese: { correct: 0, wrong: 0 },
  sentence: { completed: 0 }
}

customEmojis = {
  "alphabet|A": "🍎",
  "word|animals|Dog": "🐕",
  "chinese|lesson1|人": "👨"
}
```
