## 技术方案

### 1. 句子跟读随机 + 扩容

- 在 `startSentenceReader()` 和 `nextSentence()` 中添加随机选择逻辑
- 扩充 `sentenceData` 数据，6课各增加至8-9句，总计50句
- 新增句子需基于四五快读第一册的50个汉字组合

### 2. 进度条功能实现

- 新增 `progressState` 存储各游戏进度（总题数、正确数）
- 使用 localStorage 持久化：`localStorage.setItem('learningProgress', JSON.stringify(progressState))`
- 计算：`进度百分比 = 正确数 / Math.max(总题数, 1) * 100`
- 在 `goHome()` 时更新首页进度条显示

### 3. 图片配置功能

- 新增"配置图片"按钮（首页或游戏页顶部）
- 配置面板：列表展示所有可配置项（字母/单词/汉字）
- 每项可修改：emoji表情
- 配置数据结构：`customImages = { alphabet: {A: {emoji: '🍎'}}, word: {...}, chinese: {...} }`
- 优先使用自定义配置，无配置则使用默认值
- 使用 localStorage 持久化
