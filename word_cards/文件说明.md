# 📚 英语单词学习系统 - 文件说明

## 🎉 新增：单词图片卡片

**文件：** `单词图片卡片.docx`

**特点：**
- ✅ **横向排版** - Letter横向，节省纸张
- ✅ **每页6张卡片** - 3行×2列布局
- ✅ **统一尺寸** - 每张卡片3.3英寸宽
- ✅ **纯图片+英文** - 无中文，专注英文学习
- ✅ **彩色边框** - 方便剪裁
- ✅ **大号字体** - Emoji 120pt，单词 48pt

**适用场景：**
- 打印制作单词卡片
- 闪卡教学
- 记忆游戏
- 单词测试

## 🚀 快速使用指南

### 1. 生成单词图片卡片

```bash
python3 generate_picture_cards.py
```

**输出：** `单词图片卡片.docx` - 横向排版，每页6张卡片

### 2. 生成标准单词卡片

```bash
python3 generate_cards.py
```

**输出：** `英语单词卡片_自定义.docx` - 竖向排版，每页4张卡片

### 3. 生成短语和例句

```bash
python3 generate_phrases_optimized.py
```

**输出：** `英语短语和例句_优化版.docx` - 表格布局

### 4. 生成互动练习

```bash
python3 generate_interactive_quiz.py
```

**输出：** `英语趣味互动练习.docx` - 6个互动游戏

| 特性 | 单词图片卡片 | 标准单词卡片 |

| **文件大小** | 较小 | 较小 |

## 💡 使用建议

### 打印制作单词卡片

**推荐使用：** `单词图片卡片.docx`

**优点：**
- 横向排版，节省纸张
- 每页6张，效率更高
- 统一尺寸，整齐美观
- 彩色边框，易于剪裁

**打印设置：**
1. 纸张：Letter 或 A4
2. 方向：横向（Landscape）
3. 打印：彩色打印
4. 剪裁：沿彩色边框剪裁

---

### 学习和练习

**推荐组合：**
1. **单词图片卡片.docx** - 学习单词
2. **英语短语和例句_优化版.docx** - 学习短语例句
3. **英语趣味互动练习.docx** - 巩固练习

**学习流程：**
1. 看图片卡片，认单词
2. 读短语和例句
3. 完成互动练习
4. 生活应用
