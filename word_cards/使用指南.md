# 🎨 英语单词卡片生成系统 - 使用指南

## 📚 系统概述

这是一个为5岁儿童设计的英语单词卡片生成系统，可以：
- ✅ 管理单词数据库
- ✅ 生成精美的单词卡片（Word文档）
- ✅ 提供短语和例句
- ✅ 持续添加新单词

## 📁 文件结构

```
word_cards/
├── word_database.json          # 单词数据库（自动生成）
├── word_list_manager.py        # 单词管理模块
├── generate_cards.py           # 卡片生成脚本
├── add_new_words_example.py    # 添加新单词示例
├── images/                     # 图片文件夹
│   ├── milk.png
│   └── yogurt.png
└── 输出文档/
    ├── 英语单词卡片_自定义.docx
    ├── 英语短语和例句.docx
    └── 英语单词趣味练习题.docx
```

## 🚀 快速开始

### 1. 生成单词卡片

```bash
python3 generate_cards.py
```

这会生成两个文档：
- **英语单词卡片_自定义.docx** - 单词卡片，方便打印和剪裁
- **英语短语和例句.docx** - 每个单词的短语和例句

### 2. 查看当前单词列表

```python
from word_list_manager import print_word_list
print_word_list()
```

### 3. 添加新单词

```python
from word_list_manager import add_word

add_word(
    word="Bird",           # 英文单词
    chinese="小鸟",         # 中文翻译
    category="animals",     # 类别: animals, food, family等
    emoji="🐦",            # emoji图标
    color="#3498DB",       # 边框颜色
    bg="EBF5FB",           # 背景颜色
    difficulty=1,          # 难度等级: 1=简单, 2=中等
    phrases=[              # 简单短语（适合初学者）
        "A little bird.",
        "The bird can fly.",
        "I see a bird."
    ],
    sentences=[            # 实用例句（场景应用）
        "Look at the bird in the tree!",
        "The bird is singing.",
        "Birds have wings and can fly."
    ]
)
```

### 4. 生成特定单词的卡片

```python
from generate_cards import generate_word_cards, generate_phrases_document

# 只生成动物类单词的卡片
from word_list_manager import get_words_by_category
animals = list(get_words_by_category("animals").keys())
generate_word_cards(words=animals, output_filename="动物单词卡片.docx")
```

## 📝 单词分类

当前支持以下类别：

### 🐾 动物篇 (animals)
- Dog（小狗）、Cat（小猫）、Mouse（老鼠）
- Lion（狮子）、Leopard（豹子）、Elephant（大象）
- Bird（小鸟）、Fish（鱼）

### 🍎 食物篇 (food)
- Apple（苹果）、Banana（香蕉）、Cheese（奶酪）
- Bread（面包）、Cake（蛋糕）、Lollipop（棒棒糖）
- Milk（牛奶）、Yogurt（酸奶）、Donut（甜甜圈）

### 👨‍👩‍👧 家庭篇 (family)
- Mom（妈妈）、Dad（爸爸）

## 🎨 自定义样式

### 颜色选择建议

**动物类：**
- 暖色调：橙色、黄色、粉色
- 示例：#FF8C00（橙色）、#FFD700（金色）

**食物类：**
- 食物本身颜色：红色（苹果）、黄色（香蕉）
- 示例：#DC143C（红色）、#F4D03F（黄色）

**家庭类：**
- 温馨色调：粉色、蓝色
- 示例：#E91E63（粉色）、#2196F3（蓝色）

### Emoji选择

访问 [Emojipedia](https://emojipedia.org/) 查找合适的emoji。

## 💡 使用建议

### 1. 打印设置
- 使用彩色打印机
- A4或Letter纸张
- 沿边框剪裁成独立卡片
- 可塑封保存，重复使用

### 2. 学习方法
1. **看图说单词**：指着卡片，让孩子说出英文单词
2. **听音指图**：家长读单词，孩子找对应的卡片
3. **卡片配对**：翻卡片游戏，找相同的单词
4. **生活应用**：在生活中找到这些物品，说出英文单词

### 3. 每日学习计划
- 每天3-5个新单词
- 复习前一天学过的单词
- 使用短语和例句进行对话练习
