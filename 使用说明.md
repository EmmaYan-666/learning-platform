## 方法一：直接在手机浏览器打开（最简单）📱

### iPhone/iPad 用户：

1. **传输文件到手机**
   - 方式1：通过 AirDrop（最快）
     - 在电脑上找到文件 `word_game_v2.html`
     - 右键点击 → 共享 → AirDrop → 选择你的iPhone
     
   - 方式2：通过微信/QQ
     - 把文件发送到"文件传输助手"
     - 在手机微信中打开，点击"用其他应用打开"
     - 选择"存储到'文件'"
     
   - 方式3：通过邮件
     - 发送邮件给自己，附件包含该文件
     - 在手机邮件中下载附件

2. **在手机上打开**
   - 打开"文件"APP
   - 找到 `word_game_v2.html`
   - 点击打开 → 选择 Safari 浏览器
   
3. **添加到主屏幕（像APP一样使用）**
   - 在Safari中打开后
   - 点击底部分享按钮（方框加向上箭头）
   - 向下滑动，找到"添加到主屏幕"
   - 点击添加，命名为"英语学习"
   - 完成！桌面上会出现图标，点击即可使用

### Android 用户：

1. **传输文件到手机**
   - 通过微信/QQ传输
   - 或USB连接电脑复制到手机
   
2. **打开文件**
   - 用文件管理器找到文件
   - 用浏览器打开（Chrome/Edge等）
   
3. **添加到主屏幕**
   - 在浏览器中打开后
   - 点击菜单（三个点）
   - 选择"添加到主屏幕"

## 方法二：局域网访问（手机和电脑在同一WiFi）🌐

### 步骤：

1. **在电脑上启动服务**
   - 打开终端，输入：
     ```bash
     cd /Users/emma/WorkBuddy/20260312092945
     python3 -m http.server 8000
     ```
   - 看到提示后不要关闭终端窗口

2. **查看电脑IP地址**
   - 打开系统偏好设置 → 网络
   - 查看当前连接的WiFi
   - 记下IP地址（如：192.168.1.100）

3. **在手机上访问**
   - 确保手机连接同一WiFi
   - 打开手机浏览器
   - 输入：`http://你的IP地址:8000/word_game_v2.html`
   - 例如：`http://192.168.1.100:8000/word_game_v2.html`

4. **添加书签**
   - 在浏览器中添加书签
   - 或添加到主屏幕

## 常见问题 ❓

**Q: 语音没有声音怎么办？**
A: 
- 检查设备音量是否打开
- 检查浏览器是否允许自动播放音频
- iOS需要在设置中允许Safari访问音频

**Q: 文件在手机上打不开？**
A: 
- 确保使用浏览器打开，不是文件管理器
- iPhone推荐使用Safari
- Android推荐使用Chrome

**Q: 如何重新开始？**
A: 
- 点击"重新开始"按钮
- 或清除浏览器缓存

**Q: 错题本如何清空？**
A: 
- 点击右上角"错题本"按钮
- 点击"清空错题本"
