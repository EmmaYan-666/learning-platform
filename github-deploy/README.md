# 🎓 儿童学习游戏平台

一个为5岁儿童设计的互动学习平台，包含字母、单词、汉字和句子学习。

## ✨ 功能特性

### 📚 四大学习模块

1. **🔤 字母游戏**
   - 26个英文字母学习
   - 大小写字母配对
   - 语音发音练习

2. **📖 英语单词游戏**
   - 动物、食物、家庭等类别
   - Emoji图片选择练习
   - 即时语音反馈

3. **汉字 汉字游戏**
   - 《四五快读》第一册50个汉字
   - 拼音标注
   - 图片选择练习

4. **📝 汉字Plus - 句子跟读**
   - 50个常用句子
   - 拼音标注
   - 语音朗读（0.8倍速）

### 🎨 特色功能

- **进度追踪**：实时记录学习进度
- **图片配置**：家长可自定义emoji图片
- **随机出题**：避免机械记忆
- **语音朗读**：美式英语发音和中文朗读
- **响应式设计**：支持手机、平板、电脑

## 🚀 在线访问

访问地址：https://emmayan-666.github.io/learning-platform/

## 💻 本地运行

直接在浏览器中打开 `index.html` 文件即可。

## 📱 使用说明

1. 首页选择学习模块
2. 点击开始学习
3. 根据提示选择答案
4. 系统自动记录进度

## 🛠️ 技术栈

- 纯HTML/CSS/JavaScript
- Web Speech API（语音合成）
- LocalStorage（数据持久化）
- 无需后端服务器

## 👨‍👩‍👧 适合年龄

- 3-6岁儿童
- 英语启蒙学习
- 汉字认字练习
