# 儿童学习游戏平台 - 版本说明

## 当前版本：v2.3.1

### 🎉 最新更新（2026-03-30）

#### ✨ 新增功能

1. **语音设置面板** 🔊
   - 首页右上角添加"语音设置"按钮
   - 可调节英语语速（0.5-1.5）
   - 可调节音调（0.5-2.0）
   - 可选择特定语音
   - 新增中文句子语速调节（0.5-1.2）
   - 提供实时测试功能

2. **睿丁欣研英语课程模块** 📚
   - 课程管理（S1、S2）
   - 单词和短语整合展示
   - 视频播放功能
   - 点击发音学习

3. **课程详情页面优化**
   - 单词和短语整合到一个区块
   - 卡片网格 + 列表混合布局
   - 更清晰的学习流程

#### 🐛 Bug修复

1. **修复语音设置面板打不开的问题**
   - 使用classList替代style.display
   - 确保遮罩层正确显示

2. **修复课程详情页面数据加载问题**
   - 移除localStorage优先级
   - 确保使用最新课程数据

3. **修复英语语音发音不自然的问题**
   - 优化语音参数（语速、音调）
   - 优先选择Natural自然语音
   - 支持用户自定义调节

#### 💡 用户体验改进

- 英语语音更自然人性化
- 中文句子语速可调节（适合跟读）
- 课程学习流程更流畅
- 页面布局更紧凑合理
